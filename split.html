Chromium Split View External Protocol Handler Obstruction

Right-Click Me & Select "Open in Split View"
How to test (On vulnerable older Chromium versions):
  1. Ensure #side-by-side or #split-view flag is enabled in chrome://flags.
  2. Right-click the blue button above.
  3. Select "Open link in split view" (or side-by-side window).
  4. Observe if the external protocol prompt opens without properly showing this page's origin, or if it overlays confusingly over another active tab.
